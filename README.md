## Overview
This app serves as a comprehensive resource for understanding options trading. It covers different aspects such as options overview, pricing methods, mathematical explanations, and includes a Monte Carlo option pricing calculator.

## Features
- **Options Overview:** Get an introduction to the types and purposes of options.
- **Pricing Methods:** Learn about various methods, including the Black-Scholes model.
- **Mathematical Explanation:** Explore the mathematical formulas used in option pricing.
- **Comparison:** Compare different aspects of option pricing methods.
- **Calculator:** Utilize the Monte Carlo option pricing calculator for simulations.
- **References:** Access a list of references for further reading.

## Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd <repository-directory>
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the Streamlit app:
   ```bash
   streamlit run main.py
   ```
